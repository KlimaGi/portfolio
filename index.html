Memory app creation process

The aim is to create easier learning proces. It is based on science research.

For example one lesson has 5 specific times for repetition. Those repetition dates is counted based on calendar dates.

More over those lessons should be visible in the time line. And be sorted by them dates.

And the most interesting part is to create convininent and reusable tool. To make this program an easy to implement learning schedule.

For whoom is it? For persons who know exactly their learning topics. And those topics are enough dificult, that should be repeded in different 5 dates. And be more interested to check or proof for them self, that science reserches are correct. Result should surprise.
